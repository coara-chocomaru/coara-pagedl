# Set Appops 

su環境であればアプリ一個でappopsが出来ます。

## 概要

- ログを保存(/Android/data/com/coara/setops/files/command_logs/に保存されます。)
- appopsで一括でインストール許可

> [ネタバレ…😓]
> assetsの中のバイナリをシェルから使えばsuは不要です()

> suでappopsを設定するのが面倒な方向けです。
